# Customer Churn Prediction System

## 📌 Problem Statement
Predict telecom customer churn to help reduce revenue loss.

## 🚀 Tech Stack
- Python
- Scikit-learn
- XGBoost
- Streamlit
- SHAP

## 📊 Model Performance
ROC-AUC: 0.84

## 🌐 Web App
Interactive churn risk prediction dashboard.

## 📈 Business Impact
Helps telecom companies identify high-risk customers and take retention actions.
